## Python数据库脚本更新工具

### 配置数据库信息

​	在dist文件中找到RoutingDataImporter.exe，双击打开它，输入数据库配置的信息，需要确保输入正确的账号密码，数据库名称等信息。然后点击“保存配置”可以自动生成config.json文件，在下次打开该程序时能自动输入配置信息。

#### 增添或修改数据库表内信息

​	在dist文件中找到JSONGenerator.exe，双击打开它，可以看到上面有对于该项目的各个表名。点击需要修改的表名，在下面新增添加的数据，需要保证添加的格式正确。然后点击下面的导出json文件，再点击RoutingDataImporter.exe程序，在该程序中提交json文件，并提交修改，可自动完成数据库更新。（若对于在数据库中已有的id，则会进行更新操作，若没有则进行增添操作）
