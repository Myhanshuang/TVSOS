# README

用于arm开发板的运行脚本与其配套后端接口的设计

---

## “开发板” 文件夹

本文件夹是在arm开发板上运行的代码，用于控制arm开发板接入的modbus定位模块获取定位信息，并通过串口发送给上位机（即开发板本身）。

其中，arm_script.py是主程序，arm_script_setup.sh是用于自动化构建环境（含依赖配置）的脚本。

---

## gps_backend 文件夹

本文件夹是后端接口的代码，用于接收arm开发板经http发送过来的定位信息，并对其进行操作。

文件夹中的后端代码并未实现具体的对位置信息数据进行处理的功能（仅将其打印在了控制台上），仅提供了一个示例，便于后续与完整后端程序的对接。
